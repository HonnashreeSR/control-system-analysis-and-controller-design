# Control System Analysis and Controller Design using MATLAB

## Project Overview

This project focuses on the analysis of a control system and the design of controllers to improve system performance. The plant transfer function was analyzed using various stability and frequency response techniques. MATLAB was used to simulate system behavior and design controllers.

## Methods Used

* Pole-Zero Map
* Step Response Analysis
* Routh-Hurwitz Stability Criterion
* Root Locus Analysis
* Bode Plot Analysis
* Nyquist Plot Analysis

## Controller Design

Controllers were designed using the **Ziegler-Nichols tuning method**.

The following controllers were implemented:

* Proportional (P)
* Proportional + Integral (PI)
* Proportional + Derivative (PD)
* Proportional + Integral + Derivative (PID)

The controller parameters were tuned to achieve:

* Peak Overshoot < 10%
* Settling Time < 2 seconds
